# Terminal sorting algorithm visualiser.
## => is a small python program that demonstrate sorting algorithms speed

## *optionnal: Use [CTRL -] for n = 10.000+ 

## => How to use ?
``` bash
$> make OR make re OR make clean
$> ./sort_script
```
Select a sorting algorithm
``` bash
(1): Bubble sort
(2): Selection sort
```
Select the speed
``` bash
(1): Default
(2): Slow
(3): Very slow
```
